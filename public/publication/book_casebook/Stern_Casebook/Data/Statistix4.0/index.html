A Casebook for a First Course in Statistics and Data Analysis: Data files (Statistix)
This directory contains 38 individual Statistix 4.* formatted data files representing the data sets analyzed in the book "A Casebook for a First Course in Statistics and Data Analysis," by Samprit Chatterjee, Mark S. Handcock and Jeffrey S. Simonoff, John Wiley and Sons, New York, 1995.

The names of the files correspond to the names used in the book.

Description of data sources, and further information about the data sets, can be found in the "Descriptions of the data files" section of the book.

The files are here.
